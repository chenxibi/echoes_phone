### 关键技术点
- Tailwind 任意值类名在 CSS 中需双反斜杠转义：`bg-\[\#F2F2F7\]`
- group-hover 冒号也要转义：`group-hover\:text-black`
- SVG 图标用 `stroke` 而非 `fill`/`color`
- 使用 `[class*="..."]` 属性选择器覆盖所有类名变体
- 必须 `!important` 保证优先级
- `backdrop-filter` 需 `-webkit-` 前缀兼容 iOS
