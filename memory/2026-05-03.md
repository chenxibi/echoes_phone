### 关键技术点
- Tailwind 任意值类名在 CSS 中需双反斜杠转义：`bg-\[\#F2F2F7\]`
- group-hover 冒号也要转义：`group-hover\:text-black`
- SVG 图标用 `stroke` 而非 `fill`/`color`
- 使用 `[class*="..."]` 属性选择器覆盖所有类名变体
- 必须 `!important` 保证优先级
- `backdrop-filter` 需 `-webkit-` 前缀兼容 iOS
- `data-app-link` 属性值必须和 APP_LIST label 一致（CN 用中文，EN 用英文）
- `data-section-icon` 必须放在 h3 父元素上，不是 SVG 子元素上，否则 `::after`/`::before` 在 SVG 上无效
- `AppIcon` 组件动态绑定 `data-app-link={label}` 即可自动匹配所有 App 入口
